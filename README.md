El proyecto consiste en un programa que buscaba ser capaz de crear distintos personajes para luego ser guardados en una organizacion, desde sdonde se podia hacer diversas cosas con ellos.
Para lograr nuestro cometido creamos 3 clases principales, la de heroe, que es donde asignamos las variables que usarian nuestros personajes, siendo estas nombre, clase, nivel y vida. Como aclaracion, al basarnos en los comics de los X-Men, en la variable clase decidimos usar las distintas categoriass de mutantes existentes, y por ende todo el concepto esta orientado hacia ellos
Ademas de designar las variables y el constructor en la clase de heroe desarrollamos metodos que se apliquen de forma individual a cada personaje
Luego, creamos la clase gremio, donde se creo una lista a la que se añadiran los distitnos personajes u objetos. Ademas en esta clase creamos metodos que trabajen especificamente con la lista y lo que estaba dentro.
Por ultimo creamos una clase Main, que utilizamos como interfaz con el usuario, desde donde aparece un menu con el que el usuario es capaz de interactuar.
Ademas en la clase, lo que hicimos fue llamar a los metodos de distintas clases para presentarlos todos ahi, ademas de crear las validaciones necesarias para que el programa no sea capaz de romperse.
El menu estaba conformado por 6 opciones, donde a continuacion vamos a explicarlas.
La primer opcion llamada agregar mutantr, como es obvio permite al usuario crear y agregar nuevos objetos a la lista. PAra crear esos objetos lo que hicimos es que le pedimos al usuario que ingrese los datos mnediante el scanner con las funciones nextInt() y nextLine(), para luego llamar desde el main al metodo add que creamos en Gremio, que se encarga de agregar objetos a la lista.
En la segunda opcion lo que hicimos fue mostrar la lista de pbjetos existente en la lista, para esto recorrimos la Array y luego imprimimos el nomrbre y la clase del personaje, y luego llamamos a ese metodo desde el Main .
La tercer opcion se encarga de subir el nivel de un personaje, para hecer esto lo tuvimos que dividir en dos partes. Primero en la clase Heroe creamos la funcion base que se encarga de subir el nivel y la vida de un personaje.
Luego lo que hicimos fue crear un metodo en Gremio, ya que asi podiamos elegir a que personaje subir mediante el indice, ya que recorriamos la lista. Ademas para este metodo creamos una restriccion, y es que si el pérsonaje tenia una vida menor a igual a 0 no podia subir de nivel.
Y luego lo que hicimos en el main fue primero usar el metodo de mostrar, para que el usuario pueda ver la lista de objetos con su respectivo indice, y luego mediante un nextInt() le dimnos al usuario elegir al personaje poniendo el indice que aparecia en pantalla. y luego llamabamos al metodo subir de nivel con el indice elegido antes para subir unicamente a ese objeto.
La cuarta funcion busca verificar que personajes estan derrotados, es decir los que tengan vida negativa.Para esto desde Gremio creamos una funcion que se encargue de recorrer la lista, y que mediante un get es capaz de saber la vida que tiene el personaje. Gracias a esto exosten dos posibles resultado mediante un if, el primero, en caso donde la vida sea menor a 0 simplemente te aparece qye el personaje fue derroatado, en caso contrario te indica el nombre y vida del personaje
Para la quinta opcion decidimos calcular la fuerza. Esta opcion se divide en varias partes.
Primero, desde Heroe definimos como se calculan esos niveles, siendo en nuestro caso la ecuacion utilizada nivel*clase, donde atribuimos a clase un valor numerico desde el 2 al 64 segun la escala oficial de los mutantes, y luego utilizamos un return en cada uno de los posibles casos. 
Desde la clase Gremio lo que hicimos fue primero definir una variable vacia llamada total, para luego mediante un for recorrer la lista. Esto con el objetivo de primero devolver la fuerza individual que tienen los personajes, y para sumar constantemente esos datos a la variable total.
Luego desde el Main se hacen varias cosas, lo primero fue mediante un for, llamar y mostrar el nivel individual de cada personaje, y luego llamar al total para mostrar cuanto poder existe en total.
